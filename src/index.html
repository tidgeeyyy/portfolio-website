About Me

“Dancing faces you towards heaven, whichever direction you turn.”

Hello, I’m Thomas Joseph Almorin, a junior Information Systems student at the Technological University of the Philippines – Manila.

The first time I became aware of programming was during my high school years. My father, a teacher, would often share stories about his former students, many of whom had gone on to become successful IT graduates working both locally and abroad. He asked if I had any interest in pursuing a similar path, which made me reflect on whether this field aligned with my aspirations.

This led me to join a school club that focused on programming and related activities. It was during this time that I truly developed an interest in programming.

My Projects

Project 1: Word By Word

This is a simple C program game for 2 players that displays words one by one, and you need to memorize them as you write the whole sentence to make your answer right and proceed to the next level. This project was submitted as group final project for Computer Programming 1.

Repository: https://github.com/tidgeeyyy/word_by_word

Project 2: ATM Simulation

This ATM simulation program is developed using C++. It mimics the functionalities of an actual ATM by utilizing a flash drive as an ATM card. This program allows users to perform various transactions, including withdrawing, depositing, and transferring money. This project was submitted as group project for Data Structures and Algorithms.

Repository: https://github.com/tidgeeyyy/atm_simulation

Project 3: Text Compressor and Decompressor

This project implements a text compressor and decompressor using Huffman coding. The compressor efficiently reduces the size of text files by encoding characters based on their frequency of occurrence, while the decompressor reconstructs the original text from the compressed data. Both functionalities are developed using C++. This project was submitted as group project for Data Structures and Algorithms.

Repository: https://github.com/tidgeeyyy/text_compressor_and_decompressor
